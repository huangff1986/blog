## 文章系统

  文章的结构

  
  - 文章标题 | title        | String
  - 文章内容 | content      | String
  - 浏览次数 | viewCount    | Number
  - 评论次数 | commentCount | Number
  - 发布时间 | time         | String
  - 封面图片 | coverImg     | String
  - 文章作者 | author       | String
  - 标签类型 | tags         | Array
  - 是否发布 | isPublish    | Boolean
  - 文章简介 | desc         | String
